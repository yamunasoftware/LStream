# LStream

Video Streaming Server for Local Networks

## Build Instructions

When deploying the Docker container on your compute instance, use port 1008. The service is compatible with all major container management services, so deployment should be very straightforward.

## Media Instructions

- Only use .mp4 files as the media source
- Place all media in the /Media folder
- Make sure that all media is properly encoded
- Use Chrome, Firefox, or Edge browsers for best compatibility
